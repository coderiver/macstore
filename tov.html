Отличительные особенности

Apple Mac Pro — по праву один из мощнейших персональных компьютеров. Он инновационный, надежный и невероятно мощный. До 12 ядер чистой мощи с новыми процессорами Intel Xeon, что обеспечивает быстрое выполнение сложнейших математических вычиaслений и уверенную работу всей системы в целом.

Технология Turbo Boost позволяет динамически повышать производительность, увеличивая тактовую частоту процессора в случаях, когда приложение использует не все ядра. Технология Hyper-Threading запускает одновременно два потока на каждом ядре для обеспечения большей производительности в моменты пиковых нагрузок. Это означает, что максимальное количество виртуальных ядер может достигать 24.

Появилась возможность использования SSD-накопителей ёмкостью 512 ГБ каждый, которые можно заменять, не нарушая работу компьютера.

Комплектация

Mac Pro, кабель питания, проводная клавиатура Apple с дополнительным цифровым блоком, беспроводная сенсорная мышь Apple Magic Mouse, кабель-удлинитель USB для подключения клавиатуры, DVD-диски с программным обеспечением, документация.

В стандартную поставку также входит программное обеспечение Boot Camp, позволяющее использовать операционные системы семейства Microsoft Windows (сами системы Windows покупаются отдельно).
